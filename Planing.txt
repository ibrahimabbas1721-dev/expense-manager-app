# ✅ **Project Name:** Expense & Profit Management System

**Stack:**

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL (XAMPP)
* **Users:**

  1. **Admin** (add transactions, manage users)
  2. **User** (view-only dashboard)

---

# 🚀 **PHASE 1 — Planning & Requirements (1–2 days)**

### 🎯 Goals:

* Understand what the system should do
* Create structure & database design

### ✔️ Tasks:

1. **Write basic requirements**

   * Add expenses
   * Add income
   * Track by date, category, amount, notes
   * Admin panel for user management
   * User dashboard (read-only)

2. **Define user roles**

   * **Admin:** CRUD users + CRUD transactions
   * **User:** view transactions only

3. **Design the database** (VERY IMPORTANT FOR PORTFOLIO)

### ✔️ Database Tables:

#### **users**

| id | name | email | password | role | created_at |
| -- | ---- | ----- | -------- | ---- | ---------- |

#### **transactions**

| id | user_id | type (expense/income) | amount | category | description | date | created_at |

---

# 🚀 **PHASE 2 — Project Setup (1 day)**

### ✔️ Tasks:

1. Install & setup XAMPP
2. Setup project folder inside **htdocs**
   Example:

   ```
   C:/xampp/htdocs/expense_app/
   ```
3. Create basic file structure:

```
expense_app/
 ├── public/
 │    ├── index.php
 │    ├── login.php
 │    ├── dashboard.php
 │    └── admin/
 │          ├── users.php
 │          └── add_user.php
 ├── includes/
 │    ├── db.php
 │    ├── auth.php
 │    └── header.php
 ├── assets/
 │    ├── css/
 │    ├── js/
 │    └── images/
 └── README.md
```

4. Connect PHP to MySQL (db.php)

---

# 🚀 **PHASE 3 — Authentication System (2–3 days)**

### ✔️ Tasks:

1. Create **login page**
2. Create **registration (admin only)**
3. Create **role-based login**
4. Setup **session handling**
5. Redirect users based on roles:

   * admin → admin/dashboard.php
   * user → user/dashboard.php

Portfolios LOVE authentication projects — this adds BIG value.

---

# 🚀 **PHASE 4 — Admin Panel (4–6 days)**

### Admin Features to Build:

1. **User Management**

   * Add user
   * View all users
   * Edit/Delete users

2. **Transaction Management**

   * Add expense/income
   * Edit/Delete transactions
   * View all transactions

3. **Analytics Dashboard**

   * Total monthly income
   * Total expenses
   * Profit/Loss
   * Recent transactions table

4. **Filters**

   * Filter by date
   * Filter by category
   * Filter by amount
